HTML

  • The head element contains metadata which is information about the webpage
  • meta eletments contain information about the page that is used by the browser
  • The body element contains content that is displayed to the user
  • HTML elements are case-sensitive
  • elements are things within the opening tag to closing tag, whereas a tag refers to what is within the angle brackets

CSS

  • Inline CSS is used when a developer wants to style a particular element directly
  • Internal style CSS sheets contain rules in the head section that affects a page
  • External CSS style sheets keeps all CSS rulse in a separate file and is preferred in most cases
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

  • git checkout -b branch-name: creates a new branch and switches to it
